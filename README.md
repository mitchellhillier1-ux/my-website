# GCH Consulting — Simple Website

This is a beginner-friendly website for GCH Consulting. Edit files in any text editor and open `index.html` in your browser.

## To publish (free)
- **GitHub Pages:** upload this folder to a new repo and enable Pages (Settings → Pages).
- **Netlify:** drag-and-drop the folder at app.netlify.com.

## Enable the contact form
- Create a free form at https://formspree.io
- Replace `your-form-id` in `contact.html` with your Formspree ID.
